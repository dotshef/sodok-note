# Phase 4: 방문 완료 처리 + 고객 이력

## 목표
현장 기사가 소독 완료 체크, 약제/방법 입력. 고객별 방문 이력 타임라인 완성.

## 선행 조건
- Phase 3 완료 (캘린더, 방문 건 생성)

## 작업 목록

### 4-1. 방문 상세 페이지
- [ ] 방문 건 상세 조회 — 고객 정보, 예정일, 상태 표시
- [ ] 소독 방법 입력 (프리셋 선택: 분무/연막/훈증/기타 또는 직접 입력)
- [ ] 사용 약제 입력 (프리셋에서 선택 또는 직접 입력, 복수 선택 가능)
- [ ] 메모 입력

### 4-2. 소독 완료 처리
- [ ] "소독 완료" 버튼 → status를 `completed`로 변경, `completed_at` 기록
- [ ] 완료 시 다음 방문 예정일 자동 갱신 (schedules.next_visit_date 업데이트)
- [ ] 완료 시 다음 visit 자동 생성
- [ ] 완료 취소 기능 (admin만)

### 4-3. 고객 이력 타임라인
- [ ] 고객 상세 페이지에 방문 이력 타임라인 연결
- [ ] 날짜별 방문 기록 — 소독 방법, 사용 약제, 상태 표시
- [ ] 증명서 발급 여부 표시 (Phase 5 연동 후 활성화)

### 4-4. 기사 관리
- [ ] 기사 목록 페이지 (admin 전용)
- [ ] 기사 등록 — 관리자가 이메일/이름/비밀번호 입력하여 직접 생성
- [ ] 기사 정보 수정
- [ ] 기사 비활성화
- [ ] 방문 건에 담당 기사 배정/변경

## 산출물
- 소독 완료 체크 플로우
- 고객별 방문 이력 타임라인
- 기사 CRUD + 배정
